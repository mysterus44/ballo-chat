## 🚀 À propos du projet
**Ballo Chat** est un projet Full-Stack conçu pour offrir une expérience de discussion fluide et élégante. Conçue avec une interface utilisateur de pointe (Glassmorphism, Micro-interactions) et une architecture backend robuste pour gérer les messages en temps réel.

## 🛠️ Stack Technique
- **Frontend** : HTML5, Vanilla JavaScript, CSS3 (Architecture CSS moderne, Variables, Glassmorphism).
- **Backend** : Node.js, Express.js.
- **Temps Réel** : Socket.IO (WebSockets).
- **Base de données** : Système de base de données JSON natif (zéro dépendance externe, déploiement cloud instantané et infaillible).
- **Sécurité** : Hachage des mots de passe avec Bcrypt.

## ✨ Fonctionnalités Principales
- 💬 **Messagerie en Temps Réel** : Communication 1-à-1 instantanée grâce aux WebSockets.
- 🟢 **Statut de Présence** : Détection des utilisateurs En Ligne / Hors Ligne.
- ✍️ **Indicateur de Frappe** : Voir quand un correspondant est en train d'écrire.
- ✓ **Accusés de réception** : Statuts des messages (Envoyé, Reçu, Lu).
- 🎨 **Design Immersif** : Interface "Glassmorphism", mode sombre élégant, animations fluides.
- 🔒 **Système d'Authentification** : Inscription et connexion sécurisées.

## ☁️ Déploiement
L'application est optimisée pour être déployée facilement sur n'importe quel service Cloud (Azure App Service, Render, Alwaysdata, Heroku) grâce à son architecture sans compilation native complexe.
